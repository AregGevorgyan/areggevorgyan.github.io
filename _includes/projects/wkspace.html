An online, cloud-synchronized workspace for competitive programmers.

In competitive programming, speed is crucial, and it's easy to waste a lot of time doing repetitive tasks like making new programs from templates, creating input files, and running code on provided test data. Inspired by CS Academy's interface, this website provides an integrated problem viewer and code editor with compilation and testing built in, to eliminate repetition.

To accomplish this, the web server scrapes problems from Codeforces and displays their statements side-by-side with a code editor. It also parses sample tests and implements parallel verification on all test cases with the click of a button—no copying and pasting required. Programs are autosaved and synchronized with a MongoDB instance, so they persist across sessions and browsers. Basically, everything that a competitive programmer needs, with no local environment setup!
