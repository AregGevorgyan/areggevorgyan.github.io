A real-time, multiplayer online card game.

This is a web implementation of a real-time pattern matching card game called Set. The idea of this project originated at SPARC 2019, when I was thinking about how I could bridge the 3000 mile gap between some of the friends I'd made, after we part ways. This web application lets you play Set online with minimal overhead (no login); it's as simple as sharing a link to invite players to join your game.

In the process of designing this application, I heard good things about Firebase (a database service offered by Google), so this app ended up serving as my introduction to serverless databases, as well as an exercise in live interface design. Because it can hard for users to keep track of many simultaneous interactions, animations and colors provide visual cues to help with understandability.
