Variational autoencoder that learns latent features in handwriting.

This is a neural network that encodes 28×28 images of handwritten characters in a labeled, 6-dimensional latent space using unsupervised learning. It consists of two parts: an encoder network that puts the input through several convolutional and downsampling layers to extract latent variables, and a decoder network that takes latent variables and a label to reconstruct the original image, minimizing mean squared error. This is called an autoencoder, but to make it work as a generative model, we also add some noise to the latent variables to make it a variational autoencoder.

I implemented the model in Keras and trained it on a dataset of over 400,000 handwritten characters, and you can see the results in the web demo below, which stitches the outputs together on a canvas. Without any prior knowledge, the model is able to detect and separate out the six most important distinguishing characteristics of a person's handwriting.
