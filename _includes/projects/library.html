A competitive programmer's library of algorithms and data structures in C++.

I maintain an open-source library of about thirty-five C++ templates for advanced competitive programmers. This includes fast implementations of algorithms like FFT, Aho-Corasick, and Tarjan, as well as data structures like Link-Cut Trees and Heavy-Light Decomposition.

This is published on a website (shown in the image) using Angular and the GitHub API. It also includes a searchable collection of more than six hundred templates from other open-source competitive programming libraries.
